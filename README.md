## Lazy copy and paste the modules?


## Step 1: Generate
Npx

```sh
npx @nanopang/nqmo-codegen moduleNameA <moduleNameB moduleNameC>

// Example
npx @nanopang/nqmo-codegen admin user auth vendor
```
Global
```sh
npm i -g @nanopang/nqmo-codegen
nqmo moduleName
```
